# Security Policy

## Reporting a Vulnerability

If you find a security vulnerability in arachna, please report it privately.

Do not open a public issue.

- **GitHub:** https://github.com/dead-duke/arachna/security/advisories/new

## What to Expect

- Acknowledgment within 5 days
- Fix within 14 days
- Credit in release notes (unless you prefer anonymity)

## Security Documentation

Full threat model and security architecture are documented in
[docs/SECURITY.md](docs/SECURITY.md).
